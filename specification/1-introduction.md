# Introduction

## Purpose and Scope

GENEALOGIX defines a human-readable, version-controlled archive format for family history data.

## Design Principles

- Clarity and simplicity in file formats
- Evidence-first, assertion-aware model
- Strong provenance and traceability
- Git-native workflows and collaboration

## Terminology

- Archive: A repository of GLX entities and configuration
- Entity: A typed record such as Person, Relationship, Assertion, Source, Media
- Assertion: A discrete, source-backed claim about an entity
