# Participant Assertions Example

This example demonstrates how to represent evidence about who participated in events and relationships using participant assertions.

## Key Concepts

- **Participant Assertions**: Assertions that claim a person participated in an event with a specific role
- **Conflicting Evidence**: Multiple assertions about the same event can represent different sources with different claims
- **Roles**: Participants can have specific roles (groom, bride, witness, etc.)

## Files

- `archive.glx`: Single-file archive demonstrating participant assertions

## What This Example Shows

1. How to assert that a person participated in an event
2. How to specify roles for participants
3. How to represent conflicting evidence about participants
4. How assertions provide evidence for structural participants
