# RFC-0001: Initial Specification

- **Start Date**: 2025-01-15
- **RFC PR**: #1
- **Tracking Issue**: #1
- **Status**: Accepted

## Summary

Define the initial GENEALOGIX specification structure, schemas, and tooling.

## Motivation

Establish a foundation for interoperable genealogy archives with Git-native workflows.

## Guide-level Explanation

Provides core concept explanation, entity definitions, schemas, examples, and conformance tests.

## Reference-level Explanation

Details normative requirements for archive layout, entity formatting, and validation.

## Drawbacks

Initial scope is large; requires ongoing maintenance.

## Rationale and Alternatives

Adapts best practices from Terraform, Kubernetes, Docker Compose, and Rust RFC process.

## Prior Art

Inspired by GEDCOM and other genealogy data standards.

## Unresolved Questions

None for MVP; future iterations will refine entity attributes.

## Future Possibilities

Version 1.1 and beyond, extended entity types, and tooling automation.
